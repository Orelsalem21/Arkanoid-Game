# Arkanoid Game (Java)

A Java implementation of an Arkanoid-style arcade game, developed as part of an academic OOP project.

## Features
- Object-oriented design with clear separation of responsibilities
- Collision detection and handling
- Game loop, sprites, and listeners
- Score tracking and game events

## Technologies
- Java
- Object-Oriented Programming (OOP)
- Interfaces and Inheritance
- Event-driven design

## How to Run
Compile and run the main game class from the `src` directory.
